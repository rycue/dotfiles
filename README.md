## Add Path `.local/bin` if not  yet added in your system

Edit your `/.zshrc` (or `/.zshrc_custom`) or `/.bashrc` and append this line:
```
export PATH="$HOME/.local/bin:$PATH"
```

Save, then reload your favorite shell:

```
source ~/.zshrc
```
or 
```
source ~/.bashrc
```
